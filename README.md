# 전통주 홍보용 웹앱 프로토타입

전통주 플랫폼 사업 발표를 위한 디지털 프로토타입 웹/앱

https://ldj5098.github.io/Sool_platform/

---

## 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 기간 | 2024.02.21 ~ 2024.02.28 |
| 유형 | 1인 프로젝트 |
| 목적 | 전통주 플랫폼 사업 발표를 위한 디지털 프로토타입 개발 |

고객 요구사항을 신속하게 시각화하고, 지속적인 피드백 루프를 통해 개발 방향성을 최적화하는 것을 핵심 가치로 삼았습니다.

---

## 🛠 기술 스택

- **Frontend:** HTML, CSS, JavaScript
- **Design/Tool:** Photoshop

---

## 주요 성과

### 요구사항 시각화
모호한 구두 설명을 스케치 및 프로토타입으로 구체화하여, 개발 방향의 오차를 사전에 차단하였습니다.

### 애자일 협업
정기적인 화면 리뷰와 피드백 루프를 통해 클라이언트 요구사항을 정확히 반영하고 고객 만족도를 확보하였습니다.
